TOOL 3 — SYMBOL TRACKER + MULTIPLIER
====================================

Analyze symbols used inside text.

TRACK THESE:
@ , . < > ? / \ | { } [ ] : ; ! # $ % ^ & * ( ) _ - + = ~ ` and emoji

FEATURES:

* Detect all used symbols.
* Count how many times each symbol exists.
* Show results in cards/table.

EXAMPLE:
? → used 2 times
@ → used 5 times

ADVANCED MULTIPLIER FEATURE:
User can increase symbol count.

Example:
why?
what?

Current:
? = 2

If user adds +1:
why??
what??

Now:
? = 4

SYSTEM REQUIREMENTS:

* Modify all matching locations.
* Recalculate counts instantly.
* Live preview before apply.
* Undo support.
* Emoji-safe processing.

ADVANCED:

* Apply only to selected lines
* Apply globally
* Symbol replacement history
* Batch processing
* Symbol frequency chart
